C++11 进阶实战专栏导航

发表于 2026-03-05 分类于 技术

🚀 专栏介绍

随着现代 C++（C++11 及以上标准）的普及，深入理解和掌握现代 C++ 的核心特性、底层原理以及并发网络编程已成为优秀后端/底层开发者的必修课。

本专栏**《C++11 进阶实战》**分为 12 周，旨在从基础特性补漏到高并发网络架构，带你系统性地进行硬核突破。

📚 专栏大纲速览

本专栏涵盖以下核心模块：

  1. 现代 C++ 基础补漏：auto、decltype、右值引用、智能指针（shared_ptr, unique_ptr）。
  2. 并发与多线程：std::thread、互斥锁、条件变量、future 与 async。
  3. 无锁编程与原子操作：std::atomic 内存模型，CAS 操作深度解析。
  4. 工业级线程池实战：手写高性能、可扩展的 ThreadPool。
  5. 网络编程与异步 IO：从 Epoll 到 Boost.Asio 的底层思维模型。
  6. 全异步 TCP Server：构建百万并发的基石，架构分离与背压处理。
  7. 综合实战结业：可观测的异步 KV Server 完整项目。
